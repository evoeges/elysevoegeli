|   Mass Farmers Markets   |

Client

Mass Farmers Markets supports farmers markets throughout Massachusetts.

Problem

How can we help people easily find farmers markets near them?

Solution

We built a simple web app with toggles for the different functionalities needed by users.

Process

This project was for the MIT course Code for Good. The class partners student teams with clients. Over three weeks in January 2019, we moved rapidly through conversations with our client, background research, lo fidelity prototyping, app development, and user testing.

The project will be deployed on the new Mass Farmers Market website by the end of summer 2019.

Learnings

This project was rapidly built and deployed. With more time, we would have liked to conduct deeper user interviews with people across the state of Massachusetts looking to access farmers market services.
